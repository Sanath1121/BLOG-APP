# Blog App

Blog App is a full-stack blogging platform for registering users, publishing articles, and reading content in a clean interface.

## What It Does

- User registration and login
- Article browsing and reading
- Author publishing tools
- Profile and content management

## Project Layout

- `frontend/` — Web interface
- `backend/` — API server and database layer

## Live Deployment

- Frontend: https://blog-app-1-jrys.onrender.com
- Backend: https://blog-app-q882.onrender.com

## Quick Start

1. Install dependencies in both folders.
2. Configure backend environment variables.
3. Run the backend.
4. Run the frontend.

For setup and commands, see the folder-specific README files.
